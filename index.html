About Myself

I am Danny Harani, an aspiring software engineer and developer from Montreal, now living in Ottawa. I am in grade 12 at AY-Jackson secondary school. Computer programming has been an interest of mine for as long as I can remember and I started really looking into it when I was about 11 years old with basic block-type coding. When I was 14, I took it on to myslef to start learning to code. I used online resources, primarily youtube to learn coding on my own. I started with basic variable video's and learned up to loops. I then made simple input output type of games like hangman or rock paper scissors. When I felt confident in my abilities, I decided to learn python so I can diversify my knowledge in programming. However, I lost interest since I did not know what I should code, I had no inspiration. I took a break and decided to see what grade 10 compsci would teach me. I designed a simple game for little kids in elementary with actionscript and enjoyed the project. It boosted more of my creativity and I decided my break was over.

The year after, I learned C++ in grade 11, this programming language was more familiar to me than actionscript and I enjoyed it. This is where I am today, I am learning Javascript and the DOM through online resources like Khan academy and FreeCodeCamp. This portfolio will display 10 things I learned and why I enjoyed them. Here's a quick list of the coding languages I enjoyed learning (in no order):

  • Python
  • Javascript
  • Java
  • Kotlin
  • C++

My favorite projects

Here is a list in no particular order of my 10 favorite projects / modules:

Paint Splatter

This project is the first that really showed me the power of javascript. I used objects, functions, loops, arrays and much more to create something I believe is beautiful. It is not complex, but I feel like it was a nice introduction to what I could start doing. As a student, it taught me that most concepts of almost any course are related in some fundamental way. For example, the arrays are just a represent of a set, which we intuatively know as a group of similar "things". An array of arrays is a multiset in that same way! functions are algorithms called upon when needed, similarly to when solving a puzzle like a rubics cube. I learned a lot about clearing my misconceptions that courses are their own entities. They are all part of the same idea!

Guessing game

Games and Visualisations in Khan Academy really re-introduced me to the ability of being creative while coding. Although the concepts were all covered in previous modules, I enjoyed the 'free-ness' of this section since I believe the best way to learn is through having fun and experimenting. Games and Visualisations allowed me to do just that, experiment. Being told what to do is a very unengaging way of learning and I appreciated this new approach.

Mountain range

This specific project from natural simulations was quite fun for me, I would say the most fun from this section. The concept were complex yet easy to grasp and allowed me to use that creativity from games and Visualisations. My basic idea coming into this portion was to draw an irregular cirle using the noise but that evolved into drawing clouds with that same noise, an abstract sun with sun rays that I really had to ponder with using sin and cos. This third module shows an evolution from my first two which were much less creative and felt bland. Although games and Visualisations allowed me to show that creativity, I guess I was nervous to do so and this project helped me overcome that. However, I did come up with a complaint while programming this, I now realise I do not understand the fundamental reason why my code works since it is abstracted with sin and cos, It should be more intuitive than how it is.

Creature comforts and critter jitters

I abandoned creativity in this module to try and represent as accurately as I could the interaction of a 'fly' with its environement. The main idea is that the fly will be attracted to some food while being repelled from the predators. I believe I did a pretty decent job of doing just that, in some situations, the fly escapes the frame to not be seen again which made me happy, sometimes an environement is not suited for the creature or even a person which is why the fly would leave if it is too dangerous for it to stay. I learned a lot about how to use physics concepts while programming thanks to this module.

Interactive Slideshow with JQuery

JQuery is a great tool to personallise and make your webpage interactive. This JQuery module was a great tool to learn online and understand how to implement different kinds of libraries on your website. This section also helped me understand that you do not need to create everything from scratch and you can use other peoples work to improve your own. Now this does not mean that you can steal, but you can implement instead! I was always "iffy" on taking other people's work but if noone ever borrows others' efforts, noone would ever create anything new. I am glad this module made me understand that I can be more productive by using libraries and improving my work with them. Although I did not add too much complexity to this code, it is very expandable and modular. For example, I could use an array and have next go to next index and loop once it reaches the last and have previous do the same but for the previous index. This is just a thought but simply implementing this with that simplicity is enough to grasp the concept.

Asteroids Spaceship

This project encapsulated almost all of natural simulations and animation. I learned a lot from this experience, doing extra research online from sources like stack overflow and other communities of the sort. I believe learning is impossible without a community around you. I am glad this project pushed me outside of my comfort zone and taught me that asking questions and not understandind on the first try is completely normal. I did not ask any question on these websites since I had answers from others' asking the same questions however, I can see myself asking many, many many questions in the future. The actual project itself used many math concepts like angle and distance, I am looking for a way to implement more rational math like quadrance and spread but without success so far since I have been taught classical math for too long. This is a goal for my future!

Blog

This module, from Intro HTML and CSS sparked my passion to learn for myself. I was not quite happy with having simply met the guidlines on this one, so I went on a little bit of a online search to personallise the website to my liking. This project impacted me in two significant ways, the first being to never overlook the details and the second being to ask for help. Many of my previous misconceptions about HTML code were cleared up and this course was what triggered that for me.
